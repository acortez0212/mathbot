## Requirements

To run the bot your'll need OpenAI API key. 

Put it to `.env` file:

```bash
OPENAI_API_KEY=...
```

## Usage

Install dependencies:

```bash
pnpm i
```

Run locally:

```bash
pnpm dev
```

Deploy with SST to AWS ([How to init AWS/SST](https://sst.dev/chapters/create-an-aws-account.html)):

```bash
pnpm sst:deploy
```
